== Description ==

This plugin adds a new CSS Classes Custom Post Type for WPBakery elements. This let you create classes and apply them on WPBakery elements.
You need to install WPBakery (Visual Composer) and Advanced Custom Fields plugins before installing this plugin to be able to use these features.
